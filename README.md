# 🚀 ZbQuotes: Quotes Manager

A standalone, local-first financial quotes management system built with **Python**  and **SQLAlchemy**.


---

## ✨ Features
* **Local-First:** Uses a local SQLite database for speed and privacy.

## 🛠️ Tech Stack
* **Language:** Python 3.10+
* **Database:** SQLite
* **ORM:** SQLAlchemy
